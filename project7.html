Lie Detector Camera

An experimental system about machine judgment and behavioral control

| 项目介绍

《Lie Detector Camera》是一套实验性的交互影像系统。它通过摄像头捕捉使用者在画面中的运动幅度，并据此给出“真话/假话”的判断：当身体动作较大时，系统判定为 FALSE；当动作较小、相对静止时，系统判定为 TRUE。

这一判断逻辑是有意设计得简单且不可靠的。项目并不追求“测谎准确性”，而是将“诚实”这种复杂的人类行为，粗暴地简化为单一可量化变量，以此揭示：一个系统即使规则武断，也可以借由技术外观制造“客观判断”的幻觉。

在反馈机制上，系统采用强烈的奖惩对比：当结果为 TRUE 时，屏幕出现大规模烟花，提供正向奖励；当结果为 FALSE 时，画面被扭曲并碎片化，形成类似万花筒的混乱视觉，暗示惩罚与系统失稳。用户在这一过程中会逐渐意识到，自己正在为了迎合“机器权威”而主动调整行为。

本项目关注的核心并非真假本身，而是机器感知、行为规训与“客观真相”幻觉之间的关系。

| Project Description

Lie Detector Camera is an experimental interactive camera system that classifies whether a person is telling the truth based on physical movement in front of the camera. Large motion is labeled as FALSE, while relative stillness is labeled as TRUE.

This intentionally simplified and unreliable logic does not aim to detect real truthfulness. Instead, it reduces a complex human behavior to a single measurable variable, exposing how a system can claim objectivity while relying on arbitrary rules.

The interface uses immediate reward and punishment feedback: TRUE triggers large-scale fireworks, while FALSE produces a distorted, fragmented, kaleidoscope-like image. Through this contrast, users become more aware of how they emotionally and physically adjust themselves in response to an authoritative but flawed system.
